Welcome to our Smart Home Website!

We develop this project for the Distributed System Course. Our project name is Smart Home and we built this prototype.
Basically, we use different sensors like temperature sensor, luminosity sensor,audio sensor and so on so forth to collect data, and then send to our server. The customer from our front-end website and moble app can check these data.
